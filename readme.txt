== Description ==

This is a simple plugin that a progress bar will complete upon scrolling on your site.


== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/Progressbar-with-scrolling` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. That's all you have to do. Nothing more.


== Frequently Asked Questions ==

= What i have to do after installation?

Just activate the plugin and you are ready to go.

= What will be the effect?

You will see a progressbar at the bottom of your page while you scroll down the page.
